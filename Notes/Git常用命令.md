# Git常用命令

## 1.创建git空间

```
git init
```

## 2.删除连接的远程仓库

```
git remote remove origin
```

## 3.增加远程仓库

```
git remote add origin git@github.com:qiqizizzz/codenotes.git
```

## 4.提交到Github

```
git push -u origin master
```

## 5.查看远程仓库

```
git remote -v
```

## 6.添加

```
git add .
git commit -m 'update'
```
